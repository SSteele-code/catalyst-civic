# PRATTLE — Human Conversation Reconstruction Engine

**The cornerstone layer between PULL and PARSE in the Catalyst Civic pipeline.**

Prattle's single mandate: transform raw transcript text into clean, speaker-attributed dialogue turns while maintaining a 100% preservation mandate.

## 🏗️ Industrial Pipeline (The 5 Lobes)

1. **INGEST (`ingest.py`)**: 
   - Normalizes raw `.vtt` into uniform JSON.
   - **Industrial Squeeze**: Eliminates YouTube auto-caption redundancy.
   - **Gap Detection**: Uses 1.5s silences to ground sentence boundaries.
2. **PHONETIC TRANSLATOR (`phonetic_translator.py`)**:
   - Corrects common municipal/legal hallucinations (e.g., "Gini coefficient" -> "Virginia Code").
   - Anchors local member names and department acronyms.
3. **ROBERTS STATE (`roberts_state.py`)**:
   - **Phase Machine**: Tracks meeting progression (Call to Order -> Adjournment).
   - **Discovery Engine**: Dynamically builds the Council Roster from Roll Calls.
4. **QUOTER (`quoter.py`)**:
   - Contextual resolver for low-confidence turns.
   - Links speakers to votes and acknowledges procedural "seconds."
5. **QA (`qa.py`)**:
   - Final structural validation gate.
   - **MANDATE**: Full preservation. No disfluency removal. Everything captured by the mic is saved.

## 📂 Environment Structure

- `_Vualt/YTT/`: Source of truth for raw `.vtt` "fuel".
- `_staging/`: Temporary "engine room" for in-flight pulses (wiped on success).
- `_output/`: Final high-fidelity JSON records.
